Ferramenta para edição de dados do jogo Brasfoot usando arquivos Yaml

Um arquivo para descrição de um clube poderia ser feita a partir da seguinte forma:

```yaml
pais: 29
estado: 22
forca_inicial: 19
cor1: '#409fff'
cor2: '#000000'
id: gremio
nome: Grêmio
estadio:
  nome: Arena do Grêmio
  capacidade: 55358
tecnico:
  nome: Renato Gaúcho
  pais: 29
```
